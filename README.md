This repository contains the datasets and source code providing analytical solutions to the coding assignment and an interactive dashboard customized for users to understand and interact with world bank data on life expectancy and it's subsidiary datasets. 

To get started:

1. Clone the repository.
2. Once you enter the directory where the cloned repository exists, open the terminal and execute : setup_env.bat
3. Having the essential dependencies installed, you can now open input_paths.py file and replace the file paths with the path of datasets currently present in your local PC.
4. With this, the setup is complete.
5. Executing Statistical Analysis.py prints out the obtained results in the terminal.
6. Executing Data Visualizations.py and Dashboard.py opens the output in browser windows.
7. The results can be viewed by closing one after the other in the browser.

Have a good time running and reviewing the scripts.
